In this stage I viewed a W3Schools tutorial and learned how to create containers that flip, with one front side and a back side. I structured the element in HTML and designed it using CSS (with the flipping animation). The main logo/title of the UN sustainability goals was inserted at the top of the page. In this step I learned how to configure the width and height of an image in CSS. I also learned about div elements in HTML and how to use them to organize and configure the structure of the webpage.
